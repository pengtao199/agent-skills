# Agent Skills

This repository contains reusable agent skills.

## Skills

- `topon-ios-cocoapads`: TopOn iOS integration through CocoaPods, including setup, privacy/compliance, debugging, and Swift template assets.
- `write-development-rules`: Create, rewrite, review, or distill practical development rules and coding standards.

## Install

Install all skills:

```bash
npx skills add <owner>/<repo> -g --all
```

Install one skill:

```bash
npx skills add <owner>/<repo> -g --skill topon-ios-cocoapads -y
npx skills add <owner>/<repo> -g --skill write-development-rules -y
```
