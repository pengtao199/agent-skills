# Agent Skills

This repository contains reusable agent skills.

## Skills

- `ios-interaction-tuning`: Tune Swift iOS interaction motion, UIKit/Core Animation, UIViewPropertyAnimator, Lottie, and Rive integrations.
- `topon-ios-cocoapods`: TopOn iOS integration through CocoaPods, including setup, privacy/compliance, debugging, and Swift template assets.
- `write-development-rules`: Create, rewrite, review, or distill practical development rules and coding standards.

## Install

Install all skills:

```bash
npx skills add pengtao199/agent-skills -g --all
```

Install one skill:

```bash
npx skills add pengtao199/agent-skills -g --skill ios-interaction-tuning -y
npx skills add pengtao199/agent-skills -g --skill topon-ios-cocoapads -y
npx skills add pengtao199/agent-skills -g --skill write-development-rules -y
```
